# 隐藏蓝点设计

## 目标

在 Mac 版右键菜单中增加极简的蓝点显示控制，不影响正在进行的倒计时。

## 交互

- 桌面蓝点显示时，右键菜单显示“隐藏蓝点”。
- 点击后只隐藏桌面悬浮窗，状态栏图标和剩余时间保留。
- 蓝点隐藏时，状态栏右键菜单的同一位置改为“显示蓝点”。
- 点击“显示蓝点”后恢复悬浮窗，保留当前倒计时状态。
- 隐藏期间倒计时继续。计时结束时仍播放提示音并闪烁状态栏，但不自动显示蓝点。

## 实现

主进程使用悬浮窗当前的可见状态动态生成右键菜单项。隐藏和显示只调用 `BrowserWindow.hide()` 与 `BrowserWindow.show()`，不销毁窗口，因此渲染进程中的计时器持续运行。Windows 版也可共用该菜单行为，但本次仅重新交付 Mac 安装包。

## 测试

- 蓝点可见时生成“隐藏蓝点”命令。
- 蓝点隐藏时生成“显示蓝点”命令。
- 切换可见性不改变计时状态。
- 现有计时、拖动、状态栏与三击取消测试继续通过。
