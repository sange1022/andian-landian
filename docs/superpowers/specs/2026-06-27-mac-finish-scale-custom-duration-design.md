# Mac 结束放大与自定义时间设计

## 目标

优化 Mac 版暗淡蓝点：倒计时结束时保持现有呼吸放大动画，最大放大到原圆点的 3 倍；修复设置窗口中自定义时间无法输入和应用的问题。

## 结束动画

- 蓝点进入 `finished` 状态时，透明悬浮窗以当前中心为基准扩大，为 3 倍圆点动画留出完整空间。
- 圆点保持现有往复呼吸动画，缩放范围从 `1` 改为 `1` 到 `3`。
- 结束状态被三击取消、重置或切换时，悬浮窗恢复原始尺寸，中心位置保持不变。
- 蓝点若处于隐藏状态，计时结束时仍保持隐藏，不自动弹出。
- 提示音、状态栏闪烁、三击取消和其他计时行为保持不变。

## 自定义时间

- 用户正在编辑自定义分钟时，后台每 250 毫秒的计时状态刷新不再覆盖输入值。
- 点击 `Set` 或在输入框内按回车，都会应用自定义时间。
- 有效范围保持为 `1-180` 的整数分钟。无效值不发送计时命令，并恢复显示当前有效时长。

## 实现边界

- 窗口尺寸与位置计算放在可单元测试的纯函数中，Electron 主进程只负责应用计算结果。
- 自定义时间的解析和是否同步输入值也放在纯函数中，设置页只处理 DOM 事件。
- 版本升级为 `1.0.13`，本次只交付 Mac ARM64 安装包。

## 验证

- 测试结束窗口扩大后中心不变，恢复后回到原始边界。
- 测试自定义整数的有效范围与无效值处理。
- 测试输入框聚焦编辑时不同步后台状态。
- 运行完整现有测试，构建并校验 Mac DMG。
